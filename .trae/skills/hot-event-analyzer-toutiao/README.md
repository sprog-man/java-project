# 热点事件智能分析与邮件推送系统

## 技能介绍

一个自动化工具，用于爬取全网热点事件，运用心理学和社会学视角进行深度分析，并通过邮件发送分析报告。

## 核心功能

1. 从今日热榜（`https://tophub.today/c/news`）爬取知乎、微博、今日头条、百度、夸克五大平台的热点事件
2. 筛选总热度排名前三的事件进行深度分析
3. 以社会心理学视角撰写引人入胜的分析文章
4. 自动发送邮件到指定QQ邮箱

## 技术实现

### 脚本说明

1. **hot_event_analyzer.py**：热点事件爬取与分析脚本
   - 爬取今日热榜数据
   - 筛选热度前三的事件
   - 生成深度分析文章
   - 调用邮件发送模块

2. **email_sender.py**：邮件发送模块
   - 配置邮箱信息
   - 发送HTML格式邮件
   - 错误处理

## 环境要求

- Python 3.6+
- 依赖包：
  - requests
  - beautifulsoup4
  - lxml
  - email-validator (可选)
  - loguru (可选)

## 安装依赖

```bash
# 安装爬虫相关依赖
pip install requests beautifulsoup4 lxml

# 安装邮件发送依赖（通常已内置）
pip install --upgrade email-validator

# 可选：安装日志记录依赖
pip install loguru
```

## 使用方法

1. **配置邮箱信息**
   - 编辑 `email_sender.py` 文件，填写以下信息：
     - `SENDER_EMAIL`：发件人QQ邮箱地址
     - `AUTH_CODE`：QQ邮箱授权码（获取方式：QQ邮箱 -> 设置 -> 账户 -> 开启SMTP服务 -> 生成授权码）
     - `RECEIVER_EMAIL`：收件人邮箱地址

2. **运行脚本**
   ```bash
   python hot_event_analyzer.py
   ```

3. **查看结果**
   - 控制台输出执行过程
   - 收件邮箱收到分析报告

## 工作流程

1. 启动 `hot_event_analyzer.py` 脚本
2. 爬取今日热榜数据
3. 筛选热度前三事件
4. 对每个事件进行心理学分析
5. 生成HTML格式分析文章
6. 调用 `email_sender.py` 发送邮件
7. 输出执行结果

## 注意事项

1. 请确保网络连接正常
2. 请正确配置QQ邮箱的SMTP服务和授权码
3. 爬取频率不要过高，避免被网站封禁
4. 分析内容仅供参考，不构成任何建议

## 故障排查

- **爬取失败**：检查网络连接，或调整 User-Agent
- **邮件发送失败**：检查邮箱地址和授权码是否正确，确保SMTP服务已开启
- **分析内容为空**：检查页面结构是否变化，可能需要调整爬虫选择器
