# 公众号人文社会探寻

## 技能介绍

这是一个自动化工具，用于爬取全网与人文、社会、民生有关的热点事件，运用心理学和社会学视角进行深度分析，并通过邮件发送分析报告。

## 核心功能

- 从今日热榜（https://tophub.today/c/news）爬取知乎、微博、今日头条、百度、夸克五大平台的热点事件
- 筛选总热度排名前三的事件进行深度分析
- 撰写成引人入胜的完整文章
- 自动发送邮件到指定QQ邮箱

## 合规要求

- 严格遵守网络安全法和相关法律法规
- 禁止任何非法诱导、不良思想传播
- 文章内容需符合社会主义核心价值观
- 尊重事实，客观分析，避免主观臆断

## 技术实现

### 脚本1：热点事件爬取与分析 (hot_event_analyzer.py)

功能模块：

- **网页爬取模块**：使用 requests + BeautifulSoup 爬取热点数据，设置合理的 User-Agent 避免被反爬
- **数据筛选模块**：按总热度排序，筛选前三名事件，提取事件标题、热度值、来源平台、链接等信息
- **文章生成模块**：从社会心理学、传播学、人性洞察等视角进行深度分析，撰写成完整文章

### 脚本2：邮件发送模块 (email_sender.py)

功能要求：
- 使用 SMTP_SSL 协议连接 QQ 邮箱
- 支持 HTML 格式邮件（美观排版）
- 包含完整的错误处理机制
- 邮件主题格式：【热点分析】YYYY-MM-DD 热点事件深度解读
