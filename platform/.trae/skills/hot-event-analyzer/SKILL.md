# 热点事件智能分析与邮件推送系统

## 技能介绍

一个自动化工具，用于爬取全网热点事件，运用心理学和社会学视角进行深度分析，并通过邮件发送分析报告。

## 核心功能

1. 从今日热榜（`https://tophub.today/c/news`）爬取知乎、微博、今日头条、百度、夸克五大平台的热点事件
2. 筛选总热度排名前三的事件进行深度分析
3. 以社会心理学视角撰写引人入胜的分析文章
4. 自动发送邮件到指定QQ邮箱

## 合规要求

- ✅ 严格遵守网络安全法和相关法律法规
- ✅ 禁止任何非法诱导、不良思想传播
- ✅ 文章内容需符合社会主义核心价值观
- ✅ 尊重事实，客观分析，避免主观臆断

## 技术实现

### 脚本1：热点事件爬取与分析 (hot_event_analyzer.py)

功能模块：

1. **网页爬取模块**
   - 使用 requests + BeautifulSoup 爬取 `https://tophub.today/c/news`
   - 设置合理的 User-Agent 避免被反爬
   - 提取知乎、微博、今日头条、百度、夸克五大平台的热点数据

2. **数据筛选模块**
   - 按总热度排序，筛选前三名事件
   - 提取事件标题、热度值、来源平台、链接等信息
   - 数据结构化存储为字典列表

3. **文章生成模块**
   - 分析框架（每个事件）：
     - 社会心理学视角：群体行为分析、从众心理、社会认同理论
     - 传播学视角：信息传播路径、舆论发酵机制
     - 人性洞察：情感共鸣点、认知偏差分析
   - 写作要求：
     - 标题：吸引眼球但不夸张
     - 开头：设置悬念，引发好奇心
     - 正文：层层递进，逻辑清晰
     - 结尾：引发思考，正能量导向
   - 禁止内容：
     - 政治敏感话题
     - 暴力、色情、低俗内容
     - 虚假信息、谣言
     - 商业广告、营销内容

### 脚本2：邮件发送模块 (email_sender.py)

功能要求：

- 使用 SMTP_SSL 协议连接 QQ 邮箱
- 支持 HTML 格式邮件（美观排版）
- 包含完整的错误处理机制
- 邮件主题格式：【热点分析】YYYY-MM-DD 热点事件深度解读

## 完整工作流程

1. 启动脚本1 (hot_event_analyzer.py)
2. 爬取今日热榜数据
3. 筛选热度前三事件
4. 对每个事件进行心理学分析
5. 生成HTML格式分析文章
6. 调用脚本2 (email_sender.py)
7. 配置邮箱信息（用户填写）
8. 发送邮件到指定邮箱
9. 输出执行结果

## 依赖包安装命令

```bash
# 安装爬虫相关依赖
pip install requests beautifulsoup4 lxml

# 安装邮件发送依赖（通常已内置）
pip install --upgrade email-validator

# 可选：安装日志记录依赖
pip install loguru
```

## 使用方法

1. 配置邮箱信息（在 email_sender.py 中填写）
2. 运行 hot_event_analyzer.py 脚本
3. 系统会自动爬取热点事件、生成分析文章并发送邮件
